# Random Rock Songs Selector

A simple app that will randomly select 1 rock song, 1 metal song, and 1 famous rock quote for you!

## Installing the App

This project assumes that you have __Node.js__ installed.
Installation steps:
 1. Clone this repository or copy the raw contents and create local files for it
Optional:
 3. Open your __Terminal__ and navigate to the working directory that contains the file __package.json__ file
 4. Run `npm install`

## Running the App

 1. Open your __Terminal__ and navigate to the working directory that contains the file __app.mjs__ `cd file_path`
 2. Run the following command in your Terminal `node app.mjs`
